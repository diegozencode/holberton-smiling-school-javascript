# holberton-smiling-school-javascript
![Holberton logo](https://www.holbertonschool.com/holberton-logo.png)
> Build a web app in Javascript


## Description:bulb:
Implementing a design with bootstrap using all HTML, CSS, Accessibility, Responsive design, Bootstrap, Javascript, and jQuery knowledge.

The objective is simple: Have a fully functional web page that looks the same as the designer file.

## Project Notes:books:
* Be familiar with Figma.
* Use Bootstrap.
* Web pages witch to the tablet version when the screen width is 768px.
* Web pages switch to the mobile version when the screen width is 576px.
* Simple HTML structure.
* Use jQuery
* Javascript must be executed only when the document is loaded
